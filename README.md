# Binary_Tree_Implementation

# Insertnode :-
Adding a new Node in a Binary Tree .

# Pre_order :-
 1. Visit the root.
 2. Traverse the left subtree .
 3. Traverse the right subtree .
 
# In_order :-
   1. Traverse the left subtree .
   2. Visit the root.
   3. Traverse the right subtree .
   
 # Post_order :-
    1. Traverse the left subtree .
    2. Traverse the right subtree .
    3. Visit the root .
    
 # Level_order :-
  print nodes in level order .
